Git-it 은 Git 과 GitHub 의 기초를 가르쳐드립니다. 이 기초는 단지 초보자들만을 위한 Git 과 GitHub 스킬이 아니라, 앞으로 펼쳐질 당신의 환상적인 소셜 코딩, 버전 컨트롤과 함께하는 미래에서 계속해서 사용하게될 커맨드와 단계들입니다.

잠시

왼쪽 위 메뉴에서 간단히 원하는 언어를 선택하여 Git-it 을 전통적인 중국어, 일본어 또는 한국어로 이용할 수 있습니다.

어플리케이션 메뉴의 'Window' 은 당신을 Dictionary, Resources and About 페이지로 이동시켜 드릴것입니다.
