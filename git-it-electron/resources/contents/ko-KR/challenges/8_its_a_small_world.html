협업자(collaborator)를 프로젝트에 추가해보세요.

소셜 코딩(social coding)

다른 사람들과 같이 작업할 수 있다는 것은 GitHub 의 가장 좋은 점 중 하나입니다. 어느 장소, 어느 시간대의 누구와도 같이 작업하는 것이 훨씬 쉬워지기 때문입니다.

협업자 는 다른 사람의 저장소를 수정할 수 있도록 권한을 받은 GitHub 의 다른 유저입니다. 아마 기여자(contributors) 라는 용어를 들어봤을텐데요, 이것은 포크(fork)와 풀 리퀘스트(pull request)를 통해 프로젝트에 기여한 다른 GitHub 유저를 의미합니다.

@reporobot 는 저장소에서 협업하기 좋아하는 외계의 로봇입니다.

안녕, Repo Robot!

프로젝트에 협업자를 추가하기 위해, GitHub 계정의 저장소의 페이지에 접속해서 페이지의 위쪽에 있는 메뉴의 'Settings' 탭을 클릭하세요. 그 다음 'Collaborators' 옵션을 선택하세요. 텍스트 박스에 유저 이름을 입력하고 'Add collaborator' 를 누르세요.

GitHub 의 포크한 저장소 페이지로 가서 'reporobot'을 협업자로 추가하세요. URL 은 다음과 같아야 합니다.

github.com/유저이름/patchwork/settings/collaboration
