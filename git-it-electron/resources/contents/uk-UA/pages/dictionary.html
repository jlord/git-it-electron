Нижче наведено команди Git та терміналу які використовуються у Git-it завданнях.

Bash/Terminal Основи

  • Створити нову директорію
    • mkdir <FOLDERNAME>
  • Перейти у директорію
    • cd <FOLDERNAME>
  • Вивести вміст директорії
    • ls

Налаштування Git

  • Перевірити версію Git
    • git --version
  • Встановити своє ім’я
    • git config --global user.name "Your Name"
  • ВСтановити свій email
    • git config --global user.email youremail@example.com
  • Встановити свій Github account (чутливий до регістру)
    • git config --global user.username uSeRnAmE

Основи Git

  • Ініціалізувати Git в поточній директорії
    • git init
  • Перевірити статус репозиторія
    • git status
  • Показати зміни у файлах
    • git diff
  • Додати файли до комміту
    • git add <FILENAME>
  • Додати всі файли до коміту
    • git add .
  • Зафіксувати коміт та додати короткий опис змін які він вносить.
    • git commit -m "your commit message"
  • Копіювати віддалений репозиторій
    • git clone <URL>

Вітки Git

  • Створити нову вітку
    • git branch <BRANCHNAME>
  • Перейти у вітку
    • git checkout <BRANCHNAME>
  • Створити та одразу перейти у нову вітку
    • git checkout -b <BRANCHNAME>
  • Список усіх віток
    • git branch
  • Перейменувати поточну вітку
    • git branch -m <NEWBRANCHNAME>

Git Remotes

  • Додати віддалене з’єднання
    • git remote add <REMOTENAME> <URL>
  • Встановити URL до віддаленого з’єднання
    • git remote set-url <REMOTENAME> <URL>
  • Список віддалених з’єднань
    • git remote -v

Отримання змін Git

  • Отримати зміни
    • git pull
  • Отримати зміни з віддаленого репозиторія
    • git pull <REMOTENAME> <REMOTEBRANCH>
  • Переглянути зміни, перед їх отрмиманням
    • git fetch --dry-run

Push & Merge у Git

  • Надіслати зміни
    • git push <REMOTENAME> <BRANCHNAME>
  • Мердж вітки у поточну вітку
    • git merge <BRANCHNAME>

Видалення віддалених репозиторіїв та віток у Git

  • Видалити локальну вітку
    • git branch -D <BRANCHNAME>
  • Видалити віддалену вітку
    • git push <REMOTENAME> --delete <BRANCHNAME>
