新增一個協作者到你的專案

寫程式邊交朋友

和別人一起在 GitHub 上合作寫程式是最棒的事情了，因為 GitHub 讓來自世界各地的人一起工作變得超簡單。

協作者(Collaborators) 是被賦予特定 repository 編輯權限的使用者。你或許會聽過另一個名詞「貢獻者(Contributors)」，貢獻者指的是對一個專案透過 Fork 及發送 Pull Request 的 GitHub 使用者。

@reporobot 是喜歡對 repos 貢獻的一個外星機器人。

嗨！Repo Robot！

要新增 collaborators 到專案裡，首先，到 repository 的 GitHub 頁面，按下右邊側欄的 'Settings'（板手和螺絲起子的圖示）。選擇 'Collaborators' 分頁。輸入帳號，按 Enter 或是用滑鼠按 'Add' 就完成了喔。

去你之前 fork 過的 Patchwork repository 的 GitHub 頁面，新增一位協作者：'reporobot'。網址看起來會像是下面這樣，但 'YOURUSERNAME' 會換成你的帳號哦。

github.com/YOURUSERNAME/patchwork/settings/collaboration
