Fazendo merge local, deletando um branch e fazendo pull da upstream.

Limpando

Uma vez que você não precise mais de branches, você pode deletar eles localmente e remotamente. Isto é ótimo para evitar acumular branches antigos e sem uso.

Merge Local

Sua pull request foi unida! Seu branch foi mesclado com o branch "gh-pages" do original no GitHub. Você pode fazer merge do seu branch localmente, também.

Primeiramente, vá até o branch em que você deseja que receba as alterações. Neste caso, o branch "gh-pages".

git checkout gh-pages

Agora diga para o Git qual branch terá suas modificações adicionadas aqui — neste caso, o seu branch que começa com "add-".

git merge <NOMEDOBRANCH>

Finalizamos por deletar o branch que não mais nos serve.

git branch -d <NOMEDOBRANCH>

Você também pode deletar o branch no seu fork do GitHub:

git push <NOMEDOREMOTO> --delete <NOMEDOBRANCH>

Pull da Upstream

Por último, mas não menos importante, o original foi alterado desde que você teve sua pull request aceita — o Reporobot adicionou seu nome ao website! Se você fizer pull destas mudanças pela upstream você ficará atualizado e terá esta versão, também. Você pode ver isto ao vivo por entrar em: seuusuario.github.io/patchwork.

Para fazer pull da upstream original:

git pull upstream gh-pages
