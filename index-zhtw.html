歡迎來到 Git-it 台灣

Git-it 是一個教學 Git 和 GitHub 的終端機程式，這個網頁會給你每一個關卡的詳細教學。 就算沒有網路也是可以使用這份教學文件的唷！讓我們開始吧。

  4. GitHubbin 註冊 GitHub 帳號
  5. Remote Control 將 repository 做本機和遠端的連結
  6. Forks and Clones Fork 和 clone 一個開源的計畫
  7. Branches aren't just for Birds 建立一個 feature branch
  8. It's a Small World 邀請並和別人合作 (嗨！ @reporobot!)
  9. Pull, Never Out of Date 利用 Push 和 pull 來和 GitHub.com 同步
  10. Requesting You Pull Please 建立一個 pull request
  11. Merge Tada Merge 和刪除 branches
