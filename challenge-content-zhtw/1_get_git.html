請安裝 Git 到你的電腦上，並且設定好 Git 內部的使用者名稱和電子信箱。

Git

Git 是一個開源軟體（完全免費），作者為 Linus Torvalds，他也是 Linux 作業系統的作者。

Git 是一個專門做修改記錄的程式，在軟體、程式設計的領域中叫做版本控制。

步驟：安裝 Git

  • Windows 系統：建議你下載 GitHub Desktop 這個軟體。這個程式內包含 Git 在裡頭，而且比較容易安裝： desktop.github.com。你可以用 Git Shell 作為你的終端機。
  • Mac 系統: 你可以下載 GitHub Desktop 這個軟體，裡頭也包含 Git 的程式。 desktop.github.com（必須到「Preferences」中選擇「Install Command Line Tools」）。或是：
    • 直接下載、安裝 Git 主程式：前往 git-scm.com/downloads 並按照網頁上的步驟安裝。

Git 不像是你電腦裡的其他軟體。你不會在桌面上看到一個軟體圖示，但你可以透過終端機或是其他 Git 的電腦程式（如 GitHub Desktop）來使用。而終端機就是我們在 Git-it 中將會使用的程式！

步驟：設定 Git

安裝好 Git 後，打開 終端機（或 Bash、Shell、Prompt、命令提示字元介面），並輸入以下指令來確認你有順利安裝好 Git：

git --version

如果有安裝成功，這個指令將會告訴你，你所安裝的版本號碼，如：

git version 1.9.1

(請確定你的 git 版本是在 1.7.10以上。)

接下來，讓 Git 知道這台電腦所做的修改該連結到什麼使用者：

設定你的名字：

git config --global user.name "<Your Name>"

設定你的電子信箱：

git config --global user.email "<youremail@example.com>"
