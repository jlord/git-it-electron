The Challenge:

Gitをコンピュータにインストールして名前とメールアドレスを設定しよう。

Git

Git はオペレーティングシステムのLinuxの開発者でもあるLinux Torvaldsが作成したオープンソースソフトウェア (誰でもフリーに使える)です。

Gitは変更履歴を記録・追跡するためのソフトウェアです、プログラミング用語ではこのようなソフトウェアはバージョン管理システムと呼びます。

ステップ: Install Git

  • Windows: GitHub Desktopを利用するのをおすすめします。GitHub DesktopはGitも含まれていてインストールが簡単です: desktop.github.com. ターミナルにはGit Shellを使ってください。
  • Mac: MacもGitHub Desktopを利用できます。 Gitもインストールが出来ます desktop.github.com (Preferencesからcommand line tools installを選択してください)、 もしくは
    • Gitを単体で次のリンク先のページからダウンロードして: git-scm.com/downloads表示される手順を読んでインストールしてください。

Gitは普段コンピュータで使っているその他のプログラムと違います。デスクトップにアイコンは表示されませんが、今回Git-it!で利用するターミナルアプリやGitHub Desktop等のGitデスクトップアプリから利用が出来ます。

ステップ: Gitを設定

インストールが出来ましたらターミナルのアプリ(Git Shell、ターミナル等)を開いてください。 次のコマンドを入力してインストールができているか確認が出来ます:

$ git --version

This will return the version of Git that you're running and look something like this:

git version 1.9.1

(Any version 1.7.10 or higher is fine.)

Next, configure Git so it knows who to associate your changes to:

Set your name:

$ git config --global user.name "<Your Name>"

Now set your email:

$ git config --global user.email "<youremail@example.com>"

Tips

Dollar signs $ are often used in programming documentation to signify that the line is command line code (see the code snippets above). You don't actually type the $ in though, only type what comes after. For instance, to run the verify command above, you're actually going to type git-it verify into terminal.
