GitHub Desktop

Möglicherweise hast du GitHub Desktop schon installiert, bevor Du mit Git-it begonnen hast. Falls nicht, hier ist der Link.

Die GitHub Desktop App lässt dich vieles erledigen, was Du im Terminal mit Git machen kannst (wenn auch nicht alles!) und ist meist etwas leichter zu überblicken, als das Terminal. Ich persönlich nutze es, um größere Änderungen zu überblicken, Änderungen werden dort sehr übersichtlich dargestellt.

  • Github Desktop (OSX, Windows) Free von GitHub
  • Github Desktop Guides von GitHub

Weitere Git-Desktop-Anwendungen kannst Du Dir auf der Git Webseite anschauen: Git GUIs
