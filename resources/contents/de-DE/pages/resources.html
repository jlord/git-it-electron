GitHub Desktop

Möglicherweise hast du GitHub Desktop schon installiert, bevor Du Git-it gestartet hast. Falls nicht, hier ist der Link.

Die GitHub Desktop App lässt dich vieles erledigen, was Du im Terminal mit Git machen kannst (wenn auch nicht alles!) und ist meist etwas leichter zu überblicken, als das Terminal. Ich persönlich nutze es, um größere Änderungen zu überblicken, die Änderungen werden dort sehr gut dargestellt.

  • Github Desktop (OSX, Windows) Free by GitHub
  • Github Desktop Guides by GitHub
