GitHub Desktop

Solltest du zuvor GitHub Desktop noch nicht installiert haben, kannst du es hier herunterladen.

Die GitHub Desktop App kann in etwa dasselbe, was Git in der Konsole auch kann, einfach auf einem einfacheren Weg. Ich persönlich bevorzuge sie, um große Änderungen zu betrachten. So kannst du deine Changes noch einfacher verfolgen.
