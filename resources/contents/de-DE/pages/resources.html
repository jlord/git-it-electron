GitHub Desktop

Möglicherweise hast du GitHub Desktop schon installiert, bevor du mit Git-it begonnen hast. Falls nicht, findest du hier den Link.

Die GitHub Desktop App lässt dich vieles erledigen, was du im Terminal mit Git machen kannst (wenn auch nicht alles!) und ist meist etwas leichter zu überblicken als das Terminal. Ich persönlich nutze es, um größere Änderungen zu überblicken, da Änderungen dort sehr übersichtlich dargestellt werden.

  • Github Desktop (OSX, Windows) Free von GitHub
  • Github Desktop Guides von GitHub

Weitere Git-Desktop-Anwendungen kannst du dir auf der Git Webseite anschauen: Git GUIs
