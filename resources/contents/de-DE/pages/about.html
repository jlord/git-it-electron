Über Git-it

Git-it bringt dir die Grundlagen sowie alltägliche Git-Befehle bei, welche erfahrene Entwickler tagtäglich verwenden. Zusätzlich wird dir der Git-Flow und die Art und Weise, wie Entwickler auf GitHub zusammenarbeiten erklärt.

Einige Herausforderungen benötigen eine Internetverbindung zur Überprüfung. Die Herausforderungen können auch offline gelesen werden.

Reale Development Environment

Mit Git-it kreierst du dein eigenes Git Repository und veröffentlichst es anschließend auf GitHub. Die gesamten Herausforderungen werden im Terminal gemacht und direkt überprüft. Zum Schluss hast du deine ersten "green squares" in deinem GitHub Chart!

Code, der Git-it möglich gemacht hat

Diese App wurde mit HTML, CSS und JS gemacht. Electron wurde als zusätzliches JS Library eingebunden, um dir eine native Umgebung zu bieten. Der gesamte Code von Git-it ist online in seinem Repository auf GitHub. Git-it wurde von @jlord gemacht und anderen Contributors.
