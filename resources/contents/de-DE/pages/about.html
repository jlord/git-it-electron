Über Git-it

Git-it lehrt die essentiellen, alltäglichen Git Kommandos, die Entwickler im täglichen GitHub-Verlauf nutzen um damit auf GitHub zusammenarbeiten.

Manche Aufgaben benötigen eine Internetverbindung, um Git-it also vollständig zu absolvieren solltest Du online sein. Die Aufgaben und Informationen sind aber immer offline in der App verfügbar.

Reale Entwicklungsumbegung

Git-it benutzt das echte Terminal, Git und GitHub, anstatt diese nur zu simulieren. Wenn du Git-it beendet hast, sind Repositories auf Deinem GitHub-Account verfügbar und Deine Beitragsstatistik zeigt bereits erste Zahlen!

Der Code, aus dem Git-it besteht

Diese App ist mit HTML, CSS und JS entwickelt und benutzt die Electron Bibliothek. Jeglicher Code für Git-it ist im Repository auf GitHub online verfügbar. Git-it ist ein Dienst aus Liebe von @jlord und freundlichen Mitwirkenden.
