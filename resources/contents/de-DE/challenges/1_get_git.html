Git

Git ist eine kostenfreie Open-Source Software (der Quellcode ist frei verfügbar), die von Linus Torvalds - dem Entwickler des Linux-Kernels - geschrieben wurde.

Es handelt sich dabei um eine Software zur Versionsverwaltung, die Dateiänderungen im Verlauf der Zeit nachverfolgt und protokolliert. Wenn du bereits eine Funktion wie z.B. "Änderungen nachverfolgen" in Microsoft Word verwendet hast, ist dir dieses Konzept bereits bekannt!

Git installieren

Wir empfehlen Dir die aktuelle Version von der Git Webseite herunterzuladen und diese auf deinem Computer zu installieren.

  • Windows: Benutze Git Bash als Terminal.
  • Mac: Benutze die Terminal-App als Terminal.

Du hast Git schon installiert oder bist dir nicht sicher? Tippe git --version im Terminal ein. Wenn du in der Ausgabe eine Versions-Nummer größer als 1.7.10 siehst, ist die passende Software installiert. Weitere Informationen dazu findest du auf der Git Webseite.

Git Software

Git selbst ist kein typisches Programm auf dem Computer. Für Git selbst gibt es kein typisches Desktop-Symbol oder eine Benutzerobefläche, aber trotzdem wird Git immer für Dich verfügbar sein. Du wirst es aus dem Terminal oder durch spezielle Git-Benutzerobeflächen nutzen können.

GitHub bietet eine solche Benutzerobefläche an, die GitHub Desktop App. Die GitHub Desktop App stellt zwar einen Großteil der Funktionen von Git dem Nutzer zur Verfügung, leider jedoch nicht alle Funktionen. Daher ist es wichtig, auch die Terminal-Funktionalität von Git zu kennen. Sobald du diese aber beherrschst wird Dir die Desktop App eine nette Hilfe sein, da sie (wie die GitHub Webseite) Deine Projekte übersichtlich darstellt.

Git konfigurieren

Wenn du Git installiert hast kannst du Dein Terminal öffnen. Überprüfe die Installation von Git mit dem folgenden Befehl:

git --version

Dies gibt dir die Versions-Nummer deiner Git-Installation zurück und sollte in etwa so aussehen:

git version 1.9.1

Nun kannst du Git so konfigurieren, dass es Deine Arbeit mit Dir verknüpfen kann.

Konfiguriere Deinen Namen:

git config --global user.name "Dein Name"

Konfiguriere deine Mail-Adresse:

git config --global user.email "DeineEmail@Beispiel.de"

Damit hast du die erste Aufgabe erledigt! Klicke auf den 'Prüfen'-Button, um deine Arbeit zu kontrollieren.
