Zusammen Arbeiten macht mehr Spaß!

Das Repository, das Du bisher erstellt hast befindet sich ausschließlich auf Deinem Computer. Das ist zwar einfach, macht es aber schwer die Dateien mit Anderen zu teilen und gemeinsam daran zu arbeiten. Und genau dafür ist GitHub gemacht!

GitHub ist eine Webseite, die es Menschen aus aller Welt erlaubt Ihre mit Git online zu verwalten, damit alle gemeinsam am selben Projekt arbeiten können. GitHub fungiert dabei als ein zentrales Repository für Dich und alle, mit denen Du das Projekt teilen möchtest. Du schiebst (engl. push) Deine Änderungen auf den Server und ziehst (engl. pull) von dort die Änderungen anderer Personen.

Erstelle einen GitHub Account

Besuche github.com und registriere dich für einen kostenlosen Account. Herzlich Willkommen!

Füge Deinen GitHub Benutzernamen zu Git hinzu

Füge Deinen GitHub Benutzernamen zu Deiner Git-Konfiguration hinzu. Wir machen dies hier hauptsächlich für Git-it, nur so wird es möglich die kommenden Aufgaben zu prüfen. Speichere Deinen Nutzernamen exakt so, wie Du ihn auf GitHub erstellt hast. (Groß- und Kleinschreibung beachten! - Die Platzhalter "<" und ">" müssen nicht eingegben werden!).

Füge Deinen GitHub Benutzernamen zu Deiner Git-Konfiguration hinzu.:

git config --global user.username <USerNamE>

Was Du in deiner Git-Konfiguration eingestellt hast kannst Du nochmal überprüfen:

git config --global user.username
{{{ verify_button }}}

Stelle sicher, dass du "user.username" eingibst und nicht "user.name", welches den Namen aus der ersten Aufgabe überschreiben und Deinen GitHub Benutzernamen leer lassen würde! Falls es dir doch passiert sein sollte, wiederhole einfach die Schritte der ersten Aufgabe um Deinen Namen hinzuzufügen und wiederhole dann die Schritte aus dieser Aufgabe.

GitHub- & Git-Benutzername in der Konfiguration stimmen nicht überein.

Ein typischer Fehler ist, dass der Benutzername bei GitHub und in der Git-Konfiguration nicht übereinstimmen. Zum Beispiel ist "JLord" nicht das selbe wie "jlord"

Um Deinen mit Git gespeicherten Benutzernamen zu ändern, wiederhole einfach das ausgeführte Kommando mit dem korrekt geschriebenen Benutzernamen:

git config --global user.username <USerNamE>

Wenn Du dies wiederholt hast, kannst Du das Ergebnis erneut prüfen!
