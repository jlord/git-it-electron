Gemeinsam besser arbeiten!

Das Repository, das du erstellt hast befindet sich bisher ausschließlich lokal auf deinem Computer. Das ist zwar einfach, macht es aber schwer die Dateien mit Anderen zu teilen und gemeinsam daran zu arbeiten. Doch genau dafür gibt es GitHub!

GitHub ist eine Webseite, die es Menschen aus aller Welt erlaubt ihre Projekte mit Git online zu verwalten, damit alle gemeinsam am selben Projekt arbeiten können. GitHub fungiert dabei als ein zentrales Repository für alle, die am gemeinsamen Projekt arbeiten. Jeder Nutzer schiebt (engl. push) dabei seine eigenen Änderungen auf den Server und zieht (engl. pull) sich von dort die Änderungen anderer Personen.

Erstelle einen GitHub Account

Besuche github.com und registriere dich dort für einen kostenlosen Account. Herzlich Willkommen!

Füge deinen GitHub Benutzernamen zu Git hinzu

Füge nun deinen GitHub Benutzernamen zu deiner lokalen Git-Konfiguration hinzu. Wir benötigen dies hier hauptsächlich für Git-it, nur so wird es möglich deine Arbeit in den kommenden Aufgaben zu überprüfen. Speichere deinen Nutzernamen exakt so, wie Du ihn auf GitHub erstellt hast. (Groß- und Kleinschreibung beachten! - Die Platzhalter "<" und ">" müssen nicht mit eingegeben werden!).

Füge deinen GitHub Benutzernamen zu deiner lokalen Git-Konfiguration hinzu:

git config --global user.username <NutzerName>

Was deine Git-Konfiguration nun abgespeichert hat, kannst du auch nocheinmal überprüfen:

git config --global user.username
{{{ verify_button }}}

Stelle sicher, dass du "user.username" eingibst und nicht "user.name", welches den Namen aus der ersten Aufgabe überschreiben und deinen GitHub Benutzernamen leer lassen würde! Falls es dir doch passiert sein sollte, wiederhole einfach die Schritte der ersten Aufgabe um deinen Namen hinzuzufügen und wiederhole dann die Schritte aus dieser Aufgabe.

GitHub- & Git-Benutzername in der Konfiguration stimmen nicht überein.

Ein typischer Fehler ist, dass der Benutzername bei GitHub und in der Git-Konfiguration nicht übereinstimmen. Zum Beispiel ist "JLord" nicht das selbe wie "jlord"

Um deinen lokal mit Git gespeicherten Benutzernamen zu ändern, wiederhole einfach das ausgeführte Kommando mit dem korrekt geschriebenen Benutzernamen:

git config --global user.username <NutzerName>

Wenn du dies wiederholt hast, kannst du das Ergebnis erneut prüfen!
