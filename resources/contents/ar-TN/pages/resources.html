GitHub Desktop

Vous avez déjà probablement installé cette application avant de commencer Git-it mais si vous ne l'avez pas, voici le lien.

L'application GitHub Desktop vous permet de faire beaucoup de ce qui est faisable dans le terminal (mais pas tout). Elle est parfois un peu plus facile à utiliser et à comprendre à certains moments. Personnellement, je l'utilise pour regarder tous les diff . Vos changements sont plus simple à comprendre de cette façon.
