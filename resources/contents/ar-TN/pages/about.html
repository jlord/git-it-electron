À propos de Git-it

Git-it enseigne les commandes essentielles de Git que les développeurs utilisent tous les jours avec le GitHub Flow, le chemin emprunter par les développeurs qui collaborent sur GitHub.

Certains défis nécessitent une connexion Internet, donc vous devrez donc être en ligne pour compléter Git-it. Mais vous pouvez toujours ouvrir l'application et visualiser les défis et les ressources hors connexion.

Environnement de développement réel

Git-it utilise le terminal réel, Git et GitHub (plutôt que de les simuler). Lorsque vous avez terminé, vous aurez réellement des dépôts sur votre compte GitHub et des carrés verts sur votre tableau de contribution!

Le code qui a créé Git-it

Cette application est construite en HTML, CSS et JS et utilise la bibliothèque Electron. Tout le code de Git-it est en ligne dans son dépôt sur GitHub . Git-it, est le fruit d'un travaille fait avec amour par @jlord et tous les gentils contributeurs.
