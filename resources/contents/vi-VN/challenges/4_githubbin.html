GitHub là một website cho phép mọi người ở mọi nơi có thể đăng tải mọi thứ mà họ làm việc với Git, vì vậy mọi người đều có thể cộng tác với nhau trên cùng một project. GitHub đóng vai trò repos trung gian cho bạn và những người khác khi chia sẻ với nhau. Bạn đưa lên (push) những thay đổi của mình và kéo về (pull) những thay đổi của người khác.

Thêm mới GitHub username vào Git

Thêm mới GitHub username vào phần cấu hình Git của bạn. Ta sẽ làm việc này dành cho Git-it; nó làm cho việc xác nhận trong những challenges tới dễ dàng hơn. Hãy lưu username chính xác với tài khoản bạn đã tạo trên GitHub và nhớ viết hoa những chữ đã viết hoa. Lưu ý là, bạn không cần phải nhập "<" và ">".

Thêm mới username GitHub vào phần cấu hình Git:

git config --global user.username <USerNamE>

Kiểm tra lại việc cài đặt vừa xong:

git config --global user.username

Hãy chắc chắn rằng bạn đã gõ "user.username" trên đây chứ không phải là "user.name", điều này có thể ghi đè lên tên mà bạn đã cài đặt trong challenge đầu tiên và bỏ qua phần username! Nếu bạn nhận ra mình đã gõ sai, không sao, hãy làm lại các bước trong challenge đầu tiên và thêm tên của mình sau đó hoàn thành bước thêm username.

GitHub và Git không trùng khớp username

Một lỗi thường gặp là username GitHub của bạn không khớp với username đã được cài đặt trong phầngit config. Ví dụ, 'JLord' không giống với 'jlord'

Để đổi username đã đặt trong Git, hãy làm lại các lệnh bạn đã làm trước đó và đừng quên những phần cần viết hoa:

git config --global user.username <USerNamE>

Khi bạn đã hoàn thành cập nhật, hãy xác nhận lại nhé!
