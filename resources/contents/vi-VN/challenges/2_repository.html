Tạo một repository trên máy.

Repositories

Một repository là một bộ sưu tập của những thành phần liên quan đến nhau. Trong trường hợp ta xét đến, khi viết một phần mềm, repository là một tập hợp các files liên quan đến project phần mềm. Để dễ hình dung, repository như một bộ tài liệu của project với tất cả các files liên quan bên trong, chứa cả lịch sử thay đổi, các thành phần dự án. Có thể gọi tắt là "repos".

Hãy cho Git biết dự án của bạn, Git sẽ tự bắt đầu theo dấu tất cả sự thay đổi trong repos đó. Vì thế Git cũng tự tạo một Git repository: một folder gồm những thành phần được theo dõi bởi Git. Git theo dấu khi files đã được thêm vào, xóa đi hay thậm chí chỉ một chữ đơn lẻ trong một file bị thay đổi. Và tất cả sự thay đổi được thay đổi bởi ai, Git cũng sẽ ghi lại. Đối với lập trình phần mềm, theo dấu thay đổi như vậy được gọi là kiểm soát phiên bản - version control.

Dùng Terminal

Terminal (hay Bash) là một cách truyền đạt cho máy tính thông tin để làm việc thông qua cách dòng lệnh thay vì click trên ứng dụng bằng chuột. Bạn có thể đặt lại tên files, mở files, tạo ra folder mới, dịch chuyển giữa các directory (folder) và hơn thế nữa chỉ bằng việc chạy các dòng lệnh. Bạn có thể dùng trình soạn thảo văn bản cho code(như là Vim) trong terminal của mình.

Bên cạnh việc điều hướng cho máy tính, bạn cũng có thể sử dụng các chương trình khác Terminal mà có giao diện command-line (CLI), đồng nghĩa với việc chúng có thể chạy dòng lệnh như terminal. Git là một trong số đó. Phần đầu tiên của dòng lệnh cho máy tính hiểu rằng bạn đang giao tiếp với Git. Những phần sau là lệnh hoặc các lựa chọn khác nhau mà bạn muốn Git.

Trong Git-it, bạn sẽ học một vài lệnh cơ bản để thao tác với dòng lệnh (và cả lệnh của Git) để điều hướng cho máy tính; chúng được mô tả trong từng bước.

Tạo ra một Repository

Bạn sẽ tạo một folder cho dự án (project) và khởi tạo nó như một "kho Git" - Git repository. Ta sẽ đặt tên folder đó là 'hello-world' trong thử thách này, nhưng vẫn có thể chọn tên khác, 'hello-world' thường là tên cho project đầu tiên.

Đã đến lúc mở terminal!

Trong cửa sổ dòng lệnh Terminal, gõ những lệnh này, cùng lúc rồi enter hay return sau mỗi lệnh.

Đầu tiên, tạo một folder:
Tip: mkdir viết tắt bởi make directory - tạo directory

mkdir hello-world

Vào bên trong folder:
Tip: cd viết tắt bởi change directory - chuyển hướng

cd hello-world

Cuối cùng, nói với Git để khởi tạo (bắt đầu tracking) folder hiện tại bạn đang dùng:

git init

Dòng lệnh cuối cùng phải trả về kết quả bắt đầu với "Initialized empty Git repository". Những dòng lệnh khác không trả về kết quả.

Đã xong! Nếu bạn muốn chắc chắn rằng Git repository đã được tạo, gõ vàogit status và nếu nó không trả về'fatal: Not a git repository...', bạn đã làm đúng rồi!
