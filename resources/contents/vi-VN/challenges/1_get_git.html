Cài đặt Git trên máy tính và cấu hình nó với tên và email của bạn.

Git

Git là mộtphần mềm mã nguồn mở, miễn phí (mã nguồn được công khai) viết bởi Linus Torvalds cũng là người đã phát triển hệ điều hành kernel Linux.

Git là một chương trình theo dấu sự thay đổi theo thời gian, trong lập trình được hiểu là chương trình kiểm soát phiên bản. Nếu bạn đã sử dụng một tính năng theo dấu thay đổi trong phần mềm soạn thảo, thì bạn hiểu cơ chế rồi đó, tương tự với Git!

Install Git

Lời khuyên khi cài đặt Git trên máy tính là bạn nên download từ trang chủ ứng dụng GitHub Desktop. Ta sẽ không dùng ứng dụng desktop đối với Git-it (vì ta đang học sử dụng với terminal) nhưng Git Desktop chứa Git và là cách dễ nhất để cài đặt Git trên mọi nền tảng.

  • Windows: Sử dụng Git Shell thay thế cho terminal.
  • Mac: Chạy ứng dụng GitHub Desktop vào phần Preferences, chọn cài đặt công cụ cửa sổ dòng lệnh.Sử dụng terminal của ứng dụng như terminal trên máy.

Để chắc chắn bạn đã có Git trong máy chưa, gõ vào git --version trong terminal và kết quả trả về nếu là một con số lớn hơn phiên bản 1.7.10, bạn đã cài đặt thành công! Để tìm hiểu thêm, bạn có thể tham khảo Git website.

Git Software

GitHub Desktop có thể làm rất nhiều thứ với Git nhưng đó không phải là tất cả, đó cũng chính là lý do tại sao học Git với terminal lại quan trọng. Nhưng khi bạn có GitHub Desktop, bạn cũng sẽ cảm thấy thoải mái khi các dự án của bạn được tổ chức theo kiểu vật lý ảo giống như GitHub website.

Git không giống những chương trình bạn có trên máy tính. Bạn gần như chẳng thấy biểu tượng của Git trên màn hình desktop, nhưng nó luôn khả dụng và sẵn sàng để truy cập bất cứ lúc nào từ Terminal hoặc Git Desktop.

Configure Git

Khi GitHub Desktop (đương nhiên bao gồm Git) được cài đặt, hãy mở terminal của bạn lên. Bạn có thể xác thực Git đang khả dụng bằng cách gõ:

git --version

Kết quả trả về là phiên bản Git bạn đã cài đặt, trông sẽ như thế này:

git version 1.9.1

Tiếp theo, cấu hình cho Git để nó kết nối các phần công việc của bạn:

Đặt tên:

git config --global user.name "Ten Ban"

Và email:

git config --global user.email "youremail@example.com"
