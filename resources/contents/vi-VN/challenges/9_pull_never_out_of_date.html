Giữ file của bạn luôn được cập nhật bằng cách kéo về những thay đổi từ cộng tác viên.

Kéo về từ một Remote

Nếu bạn đang làm việc với ai đó, bạn cần phải cập nhật tất cả những thay đổi mới nhất. Vì vậy bạn sẽ phải pull về tất cả những thay đổi đã được push lên repos tại GitHub.

Reporobot đã đăng lên những gì?

Xem nếu Reporobot có bất cứ thay đổi nào đến nhánh của bạn bằng cách pull về tất cả các thay đổi từ remote origin trên GitHub:

git pull <REMOTENAME> <BRANCHNAME>

Nếu không có gì thay đổi, Git sẽ nói với bạn 'Already up-to-date'(đã được cập nhật). Nếu có thay đổi, Git sẽ nối lại những thay đổi đó vào bản cục bộ của bạn.

Reporobot có thay đổi gì không? Git nói với bạn những nơi có thay đổi. Bạn có thể mở file đó lên và xem những cập nhật từ Reporobot. Ngạc nhiên chưa, Reporobot là một nghệ sỹ!
