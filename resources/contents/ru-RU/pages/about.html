О Git-it

Git-it учит основным, повседневным командам Git, которые разработчики используют каждый день вместе с GitHub, как разработчики взаимодействуют на GitHub.

Для выполнения некоторых задач требуется подключение к Интернету, поэтому для завершения Git-it вам потребуется подключение к Интернету. Но вы всегда можете открыть приложение и просмотреть задачи и ресурсы в автономном режиме.

Реальная среда разработки

Git - он использует настоящий терминал, Git и GitHub (а не эммулирует их). Когда вы закончите, у вас будут настоящие репозитории в вашей учетной записи GitHub и зеленые квадраты на вашей диаграмме вкладов!

Код, создавший Git-it

Это приложение создано с использованием HTML, CSS и JS и использует библиотеку Electron . Весь код для Git-it находится онлайн в его репозитории на GitHub . Git - это работа любви @jlord и добрых участников.
