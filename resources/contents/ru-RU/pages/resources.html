GitHub Desktop

Скорее всего, вы установили это до того, как запустили Git-it, но если вы этого не сделали, вот ссылка.

Приложение GitHub Desktop позволяет делать большую часть того, что вы можете делать в терминале (хотя и не все), и временами его немного легче читать. Лично я использую его, чтобы просматривать когда у меня есть большие изменения . Ваши изменения действительно приятно видеть в этом виде.

  • Github Desktop (OSX, Windows) Бесплатно от GitHub
  • Руководство пользователя для Github Desktop от GitHub
