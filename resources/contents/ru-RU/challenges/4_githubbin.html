Работаем лучше вместе

Репозиторий, который вы создали до сих пор, находится только на вашем компьютере, что удобно, но делает его довольно трудным для совместного использования и работы с другими. Не беспокойтесь, GitHub предназначен для этого!

GitHub - это веб-сайт, который позволяет людям во всем мире загружать то, над чем они работают, с помощью Git, чтобы каждый мог легко работать вместе над одним проектом. GitHub действует как центральный репозиторий, которым вы и все остальные можете поделиться. Вы вносите в него свои изменения и забираете изменения у других.

Создайте учетную запись GitHub

Откройте сайт github.com и создайте бесплатную учетную запись. Дайте пять, добро пожаловать!

Добавьте имя пользователя GitHub в Git

Добавьте свое имя пользователя GitHub в конфигурацию Git. Мы сделаем это только ради Git-it; это упрощает проверку предстоящих задач. Сохраните его в том виде, в котором вы его создали, на GitHub и используйте заглавные буквы . Обратите внимание, что вам не нужно вводить "<" и ">".

Добавьте свое имя пользователя GitHub в конфигурацию Git:

git config --global user.username <Имя пользователя>

Вы можете дважды проверить, что вы установили в конфигурации Git, набрав:

git config --global user.username
{{{ verify_button }}}

Убедитесь, что вы ввели " user.username " выше, а не "user.name", которое переопределит имя, указанное вами в первом задании, и оставит вас без свойства имени пользователя! Если вы обнаружили, что сделали это, ничего страшного, просто повторите шаг в первом задании, чтобы добавить свое имя, а затем выполните шаг выше.

Имена пользователей конфигурации GitHub и Git не совпадают

Распространенной ошибкой является то, что ваше имя пользователя GitHub не совпадает с регистром, который вы задали с помощью git config . Например, «JLord» - это не то же самое, что «jlord»

Чтобы изменить имя пользователя, заданное с помощью Git, просто выполните ту же команду, что и ранее, но с правильным использованием заглавных букв:

git config --global user.username <Имя пользователя>

После внесения изменений, проверьте еще раз!
