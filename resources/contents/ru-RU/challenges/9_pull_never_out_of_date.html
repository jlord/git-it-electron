Получение изменений из удаленного репозитория

Если вы работаете над чем-то с кем-то другим, вам необходимо быть в курсе последних изменений. Поэтому вам нужно получить любые изменения, которые могли быть перенесены в центральный репозиторий GitHub.

Чем занимается Reporobot?

Проверьте, внес ли Reporobot какие-либо изменения в вашу ветку, загрузив какие-либо изменения с удаленного репозитория с именем 'origin' на GitHub:

git pull <REMOTENAME> <BRANCHNAME>

Если ничего не изменилось, появится сообщение 'Already up-to-date'. Если есть изменения, Git объединит эти изменения с вашей локальной версией.

Вносил ли Reporobot изменения? Git сообщает вам, где были внесены изменения. Вы можете открыть этот файл и увидеть обновления Reporobot. Сюрприз, Reporobot - художник!
