Объедините вашу ветку локально, удалите ветку и получите данные из отслеживаемой ветки.

Очистка

Если вам больше не нужны ветки, вы можете удалить их локально или удаленно. Это полезно, чтобы у вас не образовалась куча старых веток.

Объедините локально

Ваш запрос на принятие изменений был объединен! Ваша ветка была объединена с веткой 'gh-pages' исходного репозитория на GitHub. Вы также можете объединить свою ветку локально.

Сначала перейдите в ветку, с которой вы хотите выполнить слияние - в данном случае ветку 'gh-pages'.

git checkout gh-pages

Теперь сообщите Git, в какую ветку вы хотите выполнить объединение - в данном случае это функциональная ветка, которая начинается с "add-".

git merge <BRANCHNAME>

Приведите в порядок, удалив функциональную ветку. Теперь, когда он был объединен, он вам больше не нужен.

git branch -d <BRANCHNAME>

Вы также можете удалить ветку со своего удаленного репозитория на GitHub:

git push <REMOTENAME> --delete <BRANCHNAME>

Получите данные из отслеживаемой ветки

И последнее, но не менее важное: исходный репозиторий изменился с момента объединения вашего запроса на принятие изменений - Reporobot добавил ваше имя на веб-сайт! Если вы получите данные из отслеживаемой ветки, ваша ветка будет в актуальном состоянии. Вы также можете увидеть это вживую по адресу yourusername.github.io/patchwork.

Чтобы получить данные из отслеживаемой ветки:

git pull upstream gh-pages
