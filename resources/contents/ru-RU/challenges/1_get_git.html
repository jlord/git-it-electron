Git

Git - это бесплатное программное обеспечение с открытым исходным кодом ( исходный код открытый), написанного Линусом Торвальдсом, который также написал ядро операционной системы Linux.

Git - это программа для отслеживания изменений во времени, известная в программировании как контроль версий . Если вы использовали функцию отслеживания изменений в программе для редактирования текста, значит, вы уже знакомы с этой концепцией!

Установка Git

Мы рекомендуем установить Git на свой компьютер, загрузив GitHub Desktop. Мы не будем использовать GitHub Desktop в Git-it (мы изучаем терминал!), но оно включает Git и это самый простой способ установить Git на любых платформах.

  • Windows: Используйте Git Shell для вашего терминала.
  • Mac: Откройте GitHub Desktop и в настройках выберите установку инструментов командной строки. Используйте приложение terminal в качестве вашего терминала.

У вас уже есть Git или вы не уверены? Введите в терминале git --version , и если он вернет номер версии выше, чем 1.7.10 то все готово! Для получения дополнительной информации посетите веб-сайт Git .

Программное обеспечение Git

Приложение GitHub Desktop может делать с Git многие вещи, но не все, поэтому важно изучить терминал. Но как только вы это сделаете, вы будете рады получить настольное приложение, потому что оно организует информацию вашего проекта более наглядно, как веб-сайт GitHub.

Git сам по себе не похож на другие программы на вашем компьютере. Скорее всего, вы не увидите значок на рабочем столе, но он всегда будет доступен вам, и вы сможете получить к нему доступ в любое время из своего терминала или настольных приложений Git.

Настройте Git

После установки GitHub Desktop (и следовательно, Git) откройте свой терминал . Вы можете убедиться, что Git действительно установлен, набрав:

git --version

Это вернет версию Git на вашем компьютере и будет выглядеть примерно так:

git version 1.9.1

Затем настройте Git, чтобы он знал, как связать вашу работу с вами:

Укажите свое имя:

git config --global user.name "Your Name"

Теперь укажите вашу электронную почту:

git config --global user.email "youremail@example.com"

Вы закончили вашу первую задачу! Нажмите кнопку "Подтвердить", чтобы проверить ее.

Знак доллара в коде документации

Знаки доллара $ часто используются в документации по программированию для обозначения того, что строка является командой строкой (см. фрагменты кода выше). На самом деле вы не вводите $ , а набираете только то, что будет после.
