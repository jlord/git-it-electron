Социальное кодирование

Работа с другими людьми - одна из лучших вещей в GitHub, потому что оно упрощает работу из любого места или часового пояса.

Соавторы (Collaborators) - это другие пользователи GitHub, которым разрешено вносить изменения в репозиторий, принадлежащий другому лицу. Вы также можете услышать термин соучастник (contributor) , который описывает другого пользователя GitHub, который внес вклад в проект с помощью форка и запроса на принятие изменений (Pull Request).

@reporobot робот из космоса, который любит сотрудничать в репозиториях.

Привет, Repo Robot!

Чтобы добавить соавторов в один из ваших проектов, посетите страницу репозитория в своей учетной записи GitHub и щелкните вкладку «Настройки» в меню в верхней части страницы. Затем выберите вариант «Соавторы». Введите имя пользователя в текстовое поле и нажмите кнопку "Добавить".

Перейдите на страницу вашего разветвленного репозитория Patchwork на GitHub и добавьте 'reporobot' в качестве соавтора. URL-адрес должен выглядеть так, но с вашим именем пользователя.

github.com/YOURUSERNAME/patchwork/settings/collaboration
