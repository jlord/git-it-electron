Voici les commandes de terminal et Git utilisées dans les défis de Git-it.

Bash/Terminal Basics

  • Créer un nouveau répertoire
    • mkdir <FOLDERNAME>
  • Naviguer à l'interieur d'un répertoire
    • cd <FOLDERNAME>
  • Lister le contenue d'un répertoire
    • ls

Configuring Git

  • Vérifier la version de Git
    • git --version
  • Configurer votre nom
    • git config --global user.name "Your Name"
  • Configurer votre email
    • git config --global user.email youremail@example.com
  • Configurer votre compte Github (case sensitive)
    • git config --global user.username uSeRnAmE

Git Basics

  • Activer Git pour un dossier
    • git init
  • Vérifier l'état des changements dans un dossier
    • git status
  • Affichier les changements des fichiers
    • git diff
  • Ajouter les changements d'un fichier au prochain commit
    • git add <FILENAME>
  • Ajouter tous les changements
    • git add .
  • Pour soumettre (aka commit) les changements ajoutés avec un message de description
    • git commit -m "your commit message"
  • Copier un dépôt en local
    • git clone <URL>

Git Branches

  • Créer une nouvelle branche
    • git branch <BRANCHNAME>
  • Se déplacer sur une branche
    • git checkout <BRANCHNAME>
  • Créer et se déplacer sur la nouvelle branche
    • git checkout -b <BRANCHNAME>
  • Lister les branches
    • git branch
  • Renommer la branche courante
    • git branch -m <NEWBRANCHNAME>

Git Remotes

  • Ajouter un remote
    • git remote add <REMOTENAME> <URL>
  • Configurer l'URL vers un remote
    • git remote set-url <REMOTENAME> <URL>
  • Lister les remotes
    • git remote -v

Pulling in Git

  • Pull (tirer) les derniers changements
    • git pull
  • Pull les changement depuis une branche sur un remote
    • git pull <REMOTENAME> <REMOTEBRANCH>
  • Voir les changement avant de les tirer effectivement
    • git fetch --dry-run

Push & Merge in Git

  • Push (pousser) les changements soumis
    • git push <REMOTENAME> <BRANCHNAME>
  • Merge une branche dans la branche courante
    • git merge <BRANCHNAME>

Deleting Remotes and Branches in Git

  • Suprimer une branche locale
    • git branch -D <BRANCHNAME>
  • Suprimer une branche distante
    • git push <REMOTENAME> --delete <BRANCHNAME>
