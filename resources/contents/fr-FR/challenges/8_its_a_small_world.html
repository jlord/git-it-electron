Social Coding

Travailler avec les autres est l'une des meilleures fonctionnalité de GitHub. Grâce à cet outil, il est plus simple de travailler avec quelqu'un d'autre quelque soit l'endroit ou le fuseau horaire.

Les collaborateurs sont d'autres utilisateurs de GitHub autorisés à effectuer des modifications dans un dépôt appartenant à quelqu'un d'autre. Vous pouvez également voir le terme contributeurs, il désigne un autre utilisateur GitHub qui a apporté des contributions, par le biais d'un fork et de 'Pull Request'.

@reporobot est un robot de l'espace extra-atmosphérique qui aime collaborer sur les repos.

Hello, Repo Robot!

Pour ajouter des collaborateurs à l'un de vos projets, visitez la page du dépôt sur votre compte GitHub et cliquez sur l'onglet "Settings" dans le menu de la partie supérieure de la page. Ensuite, sélectionnez l'option 'Collaborators'. Tapez le nom d'utilisateur dans la zone de texte ajouter et cliquez sur 'Add'.

accédez à la page de votre dépôt forké 'patchwork' sur GitHub et ajoutez 'reporobot' en tant que collaborateur. L'URL devrait ressembler à ceci, mais avec votre nom d'utilisateur.

github.com/YOURUSERNAME/patchwork/settings/collaboration
