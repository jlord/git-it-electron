Mai jos sunt comenzile de terminal și Git folosite în provocările Git-it.

Fundamente de bash/terminal

  • Creează un nou folder (creează director)
    • mkdir <NUME_FOLDER>
  • Treci într-un folder existent (schimbă director)
    • cd <NUME_FOLDER>
  • Listează elementele dintr-un folder
    • ls

Configurare Git

  • Verifică versiune Git
    • git --version
  • Setează-ți numele
    • git config --global user.name "numele tău"
  • Setează-ți adresa de email
    • git config --global user.email adresa_ta@example.com
  • Setează-ți contul de Github (sensibil la litere mari)
    • git config --global user.username nUmEuTiLiZaToR

Fundament Git

  • Activează Git pentru un folder
    • git init
  • Verifică starea schimbărilor la un depozit
    • git status
  • Vizualizează schimbări pentru fișiere
    • git diff
  • Adaugă schimbările unui fișier pentru a fi operate
    • git add <NUME_FIȘIER>
  • Pentru a adăuga toate schimbările la fișiere
    • git add .
  • Pentru a opera (salva) schimbările pe care le-ai adăugat cu un scurt mesaj care descrie schimbările
    • git commit -m "mesaj de operare"
  • Copiază un depozit pe calculatorul tău
    • git clone <URL>

Ramuri Git

  • Creează o nouă ramură
    • git branch <NUME_RAMURĂ>
  • Treci într-o ramură
    • git checkout <NUME_RAMURĂ>
  • Poți crea și trece într-o ramură cu un rând
    • git checkout -b <NUME_RAMURĂ>
  • Listează ramurile
    • git branch
  • Redenumește ramura în care te afli acum
    • git branch -m <NUME_RAMURĂ>

Depozite Git la distanță

  • Adaugă conexiuni la distanță
    • git remote add <NUME_DEPOZIT_LA_DISTANȚĂ> <URL>
  • Setează URL către depozit la distanță
    • git remote set-url <NUME_DEPOZIT_LA_DISTANȚĂ> <URL>
  • Vizualizează conexiuni la distanță
    • git remote -v

Retrage în Git

  • Retrage schimbări
    • git pull
  • Retrage schimbări de pe o ramură la distanță
    • git pull <NUME_DEPOZIT_LA_DISTANȚĂ> <NUME_RAMURĂ_LA_DISTANȚĂ>
  • Vizualizează schimbările la depozit înainte să retragi
    • git fetch --dry-run

Împinge și fuzionează în Git

  • Împinge schimbări
    • git push <NUME_DEPOZIT_LA_DISTANȚĂ> <NUME_RAMURĂ_LA_DISTANȚĂ>
  • Fuzionează o ramură în ramura curentă
    • git merge <NUME_RAMURĂ>

Șterge depozite la distanță și ramuri în git

  • Șterge o ramură locală
    • git branch -D <NUME_RAMURĂ>
  • Șterge o ramură la distanță
    • git push <NUME_DEPOZIT_LA_DISTANȚĂ> --delete <NUME_RAMURĂ>
