Despre Git-it

Git-it te învață comenzile esențiale, comune din Git pe care le folosesc dezvoltatorii zi de zi împreună cu GitHub, modul în care colaborează dezvoltatorii pe GitHub.

Unele provocări necesită conexiune la internet, așa că trebuie să fii conectat pentru a termina Git-it. Dar poți oricând deschide aplicația și vedea provocările și resursele fără a fi conectat.

Mediu real de dezvoltare

Git-it folosește terminalul real, Git și GitHub (în loc să le emuleze). Când termini vei avea depozite reale în contul GitHub și pătrate verzi în graficul de contribuții!

Codul care a creat Git-it

Această aplicație este construită cu HTML, CSS și JS și folosește biblioteca Electron. Tot codul pentru Git-it este online în depozitul din GitHub. Git-it este produsul din pasiune al lui @jlord și al contribuitorilor.
