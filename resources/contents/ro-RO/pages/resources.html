GitHub Desktop

Probabil l-ai instalat îainte să începi Git-it, dar, dacă nu ai făcut-o, iată linkul.

Aplicația GitHub Desktop îți permite să faci mare parte din ceea ce poți face în terminal (deși nu tot) și este puțin mai ușor de citit uneori. Personal, o folosesc să mă uit peste orice diff care nu este mic. Schimbările pe care le faci sunt foarte frumos de văzut așa.
