Fuzionează-ți ramura locală, șterge ramura și retrage de la 'upstream'.

Curățarea

Odată ce nu mai ai nevoie de ramuri, poți să le ștergi local și la distanță. Asta ajută ca să nu sfârșești cu o grămadă de ramuri vechi.

Fuzionează local

Cererea ta de retragere a fost fuzionată! Ramura ta a fost fuzionată în ramura 'gh-pages' de pe GitHub originar. Îți poți fuziona și ramura locală.

Întâi, treci pe ramura în care vrei să fuzionezi, în acest caz, ramura 'gh-pages'.

git checkout gh-pages

Acum spune-i lui Git ce ramură vrei să fuzionezi, în acest caz, ramura cu opțiuni care începe cu „add-”.

git merge <NUMERAMURĂ>

Fă curat ștergând ramura cu opțiuni. Acum că a fost fuzionată nu mai ai nevoie de ea.

git branch -d <NUMERAMURĂ>

Poți de asemenea să ștergi ramura din depozitul la distanță de pe GitHub:

git push <NUMEDEPOZITLADISTANȚĂ> --delete <NUMERAMURĂ>

Retrage de pe upstream

Și ultimul dar nu cel din urmă, versiunea originară s-a schimbat de când cererea ta de retragere a fost fuzionată, Reporobot ți-a adăugat numele la site! Dacă retragi aceste schimbări de pe upstream vei fi la zi și vei avea de asemenea acea versiune. Poți, de fapt, să o vezi activă la: numeletăudeutilizator.github.io/patchwork.

Pentru a retrage din depozitul „upstream” originar:

git pull upstream gh-pages
