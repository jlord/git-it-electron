Adaugă numele de utilizator de GitHub în Git

Adaugă-ți numele de utilizator de GitHub în configurația de Git. Vom face asta doar de dragul lui Git-it; este mai ușor să verificăm provocările următoare. Salvează-l exact cum l-ai creat pe GitHub și folosește litere mari unde este cazul. Nu trebuie să adaugi "<" și ">".

Adaugă-ți numele de utilizator GitHub la configurația de Git:

git config --global user.username <NUmeUTilIZatOR>

Poți verifica ce ai setat în Git config tastând:

git config --global user.username
{{{ verify_button }}}

Asigură-te că tastezi "user.username" mai sus, nu "user.name", ceea ce ar suprascrie numele pe care l-ai setat în prima provocare și te va lăsa fără proprietate username! Dacă descoperi că ai făcut asta, este ok, repetă pasul din prima provocare pentru a-ți adăuga numele și apoi termină pasul de mai sus.

Numele de utilizator de GitHub și Git config nu se potrivesc

O eroare comună este să nu ai numele de utilizator de GitHub identic cu cel pe care l-ai setat cu git config. De exemplu, 'JLord' nu este același ca 'jlord'

Pentru a-ți schimba numele de utilizator setat cu Git, execută aceeași comandă pe care ai executat-o mai devreme, dar cu litere mari corecte:

git config --global user.username <NUmeUTilIZatOR>

După ce ai actualizat, verifică din nou!
