Depozite la distanță

Când pui ceva pe GitHub, acea copie rămâne pe unul dintre serverele GitHub. Aceasta îl face un depozit la distanță, deoarece nu este pe calculatorul tău, ci pe un server „la distanță” și altundeva. Împingându-ți schimbările locale (de pe calculator) pe el, îl menții actualizat.

Alții pot mereu să obțină ultimele schimbări din proiectul tău retrăgându-ți schimbările de pe depozitul la distanță (pe calculatorul tău). Acesta este modul în care toți pot lucra la un proiect fără a avea nevoie de acces la calculatorul tău unde este stocată copia ta locală.

Creează un depozit la distanță

Vrei să îți sincronizezi versiunea locală cu cea stocată pe GitHub.com. Mai întâi, creează un nou depozit la distanță pe GitHub.com.

  • Du-te la github.com, autentifică-te și fă click pe '+' în dreapta sus și apoi clic pe 'New repository'.
  • Alege un nume care se protrivește cu numele depozitului local, 'hello-world', și o scurtă descriere.
  • Fă-l public. Asta înseamnă că va fi listat în profilul public.
  • Nu-l inițializa cu un README, deoarece avem deja un fișier local numit 'readme.txt'. Aceasta este o opțiune ajutătoare de la GitHub dacă nu l-ai creat deja.
  • Lasă '.gitignore' și 'license' setate la 'none'. Nu le vom folosi în acest tutorial.
  • Fă click pe 'create repository'!

Fișiere readme, .gitignore și licențe

Acestea sunt fișiere comune în proiectele open source, așa că, pentru a face începutul unui proiect mai ușor, GitHub oferă o opțiune pentru crearea lor pentru tine.

Un readme explică ce este proiectul, cum să îl folosești și, adesea, cum să contribui (deși, uneori, există un fișier suplimentar, 'CONTRIBUTING.md', pentru acele detalii).

Un .gitignore este o listă de fișiere pe care Git nu ar trebui să le urmărească, de exemplu, fișiere cu parole!

Un fișier de licență este un tip de licență pe care o pui pe proiectul tău. Acesta le permite altora să știe cum îl pot folosi. Informații despre tipuri sunt aici: choosealicense.com.

Conectează-ți depozitul local la cel la distanță

Acum ai un depozit gol început pe GitHub.com. În partea de sus vei vedea 'Quick Setup', asigură-te că butonul 'HTTPS' este selectat și copiază adresa—aceasta este adresa depozitului tău de pe serverele GitHub.

Înapoi în terminal și în interiorul folderului 'hello-world' pe care l-ai inițializat ca depozit Git în provocarea anterioară, vrei să îi spui lui Git să își amintească adresa versiunii la distanță de pe serverele GitHub. Poți avea multiple depozite la distanță, așa că fiecare necesită un nume. Depozitul la distanță primar se numește tipic origin.

Pentru a adăuga un depozit la distanță 'origin' depozitului tău:

git remote add origin <URLDELAGITHUB>

Depozitul local știe acum unde se află depozitul tău la distanță numit 'origin', pe serverele GitHub. Privește-o ca adăugând un nume și o adresă în contactele preferate; acum, când trebuie să trimiți ceva acolo, poți.

Utilizatorii de Windows:

Dacă ai GitHub Desktop pe calculator, un depozit la distanță 'origin' este creat automat în depozitul local. În acel caz, va trebui să îi spui ce URL să asocieze cu 'origin'. Folosește această comandă în loc de cea cu 'add' de mai sus:

git remote set-url origin <URLDELAGITHUB>

Împinge-ți schimbările în depozitul la distanță

Vrei în continuare să împingi (trimiți) tot ce ai făcut local în depozitul la distanță de pe GitHub. Asta este ceva ce vei face adesea, astfel încât versiunea la distanță să fie la zi și să reflecte starea versiunii locale.

Git are un sistem de ramificare, astfel încât să poți lucra la diferite părți ale unui proiect la momente diferite. Vom învăța mai multe despre asta mai târziu, dar, implicit, prima ramură se numește 'master'. Când împingi (și mai târziu retragi) pe un proiect, îi spui lui Git numele ramurii pe care o vrei și numele depozitului la distanță pe care se află.

În acest caz, ne vom trimite ramura numită 'master' pe depozitul la distanță de pe GitHub numit 'origin'.

git push origin master

Acum du-te pe pagina depozitului la distanță de pe GitHub.com și reîncarcă pagina. Oau! Totul este acum la fel local cum este la distanță. Felicitări pentru primul tău depozit public!
