Git

Git este un soft cu surse deschise gratui (codul sursă este public) scris de Linus Torvalds, care a scris și nucleul sistemului de operare Linux.

Git este un program pentru a ține evidența schimbărilor în timp, cunoscut în programare ca control al versiunii. Dacă ai folosit o funcție de urmărire a schimbărilor într-un editor text, atunci ești deja obișnuit cu conceptul!

Instalează Git

Îți recomandăm instalarea Git pe calculator prin descărcarea aplicației GitHub Desktop. Nu vom folosi aplicația desktop în Git-it (învățăm terminalul!) dar include Git și este modul cel mai ușor de a instala Git pe toate platformele în același fel.

  • Windows: Folosește Git Shell pentru terminal.
  • Mac: Deschide GitHub Desktop și de la Preferences, selectează command line tools install. Folosește aplicația de terminal ca terminal.

Ai deja Git sau nu ești sigur? Scrie git --version în terminal și dacă întoarce un număr de versiune mai mare decât 1.7.10, ești pregătit! Pentru mai multe informații, vizitează website-ul Git.

Softul Git

Aplicația GitHub Desktop poate face multe lucruri cu Git, dar nu pe toate, motiv pentru care învățarea terminalului este importantă. Dar odată ce o stăpânești vei fi fericit să ai aplicația de desktop, deoarece îți organizează informația proiectului mai vizual, ca site-ul GitHub.

Git singu nu este ca alte programe de pe calculatorul tău. Probabil nu vei vedea o iconiță pe desktop, dar va fi mereu disponibil și vei putea să îl accesezi oricând de la terminl sau din aplicațiile Git de desktop.

Configurează Git

Odată ce GitHub Desktop (și, prin urmare, Git) este instalat, deschide-ți terminalul. Poți verifica dacă Git chiar este prezent tastând:

git --version

Aceasta va întoarce versiunea de Git de pe calculator și va arăta cam așa:

git version 1.9.1

În continuare, configurează Git ca să știe să îți asocieze contribuția cu tine:

Setează-ți numele:

git config --global user.name "Numele Tău"

Acum setează-ți adresa de email:

git config --global user.email "adresatadeemail@example.com"

Ai terminat prima provocare! Fă click pe butonul 'Verifică' pentru a valida provocarea.
