Pobieranie z repozytorium zdalnego

Jeśli pracujesz nad czymś z kimś innym musisz posiadać aktualną wersję projektu. Będziesz chciał pobrać zmiany które mogły zostać wypchnięte do centralnego repozytorium GitHub.

Co zrobił Reporobot?

Sprawdź czy Reporobot dokonał zmian na Twojej gałęzi próbując pobrać zmiany z 'origin' na GitHub:

git pull <REPOZYTORIUMZDALNE> <NAZWAGAŁĘZI>

Jeśli nic się nie zmieniło, otrzymasz komunikat 'Already up-to-date'. Jeśli jednak pojawiły się zmiany, Git scali je do Twojej wersji lokalnej.

Czy Reporobot zrobił jakieś zmiany? Git powie Ci gdzie zostały one zrobione. Możesz otworzyć ten plik i zobaczyć poprawki Reporobota. Niespodzianka, Reporobot jest artystą!
