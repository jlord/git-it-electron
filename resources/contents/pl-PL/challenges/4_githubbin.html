Utworzenie konta GitHub i dodanie nazwy użytkownika do konfiguracji Gita.

Lepsza (współ)praca

Repozytorium które utworzyłeś znajduje się w tej chwili tylko na Twoim komputerze, co jest poręczne, ale czyni to je niezbyt wygodnym gdy chcesz współpracować z innymi. Nie martw się, właśnie po to istnieje GitHub!

GitHub jest stroną pozwalającą ludziom z całego świata na udostępnianie tego nad czym obecnie pracują używając Gita, by każdy mógł wygodnie współpracować nad tym samym projektem. GitHub działa jak centralne repozytorium dla Ciebie i innych. 'Wypychasz' (od angielskiego push) swoje zmiany i pobierasz (od angielskiego pull) zmiany od innych osób.

Tworzenie konta GitHub

Odwiedź stronę github.com i zarejestruj swoje darmowe konto. Witamy, przybij piątkę!

Dodawanie nazwy użytkownika GitHub do Gita

Dodaj swoją nazwę użytkownika GitHub do konfiguracji Gita. Zrobimy to na potrzeby Git-it; ułatwi to weryfikację przyszłych wyzwań. Zapisz ją dokładnie w taki sposób w jaki widnieje ona w GitHub i pamiętaj, iż wielkość znaków ma znaczenie. Zauważ, że nie musisz wpisywać znaków "<" i ">".

Dodaj nazwę użytkownika GitHub do swojej konfiguracji Gita:

git config --global user.username <NazwaUżytkownika>

Zawsze możesz upewnić się co do poprawności ustawień wpisując:

git config --global user.username
{{{ verify_button }}}

Upewnij się, że powyżej wpisujesz "user.username" a nie "user.name", które nadpisałoby ustawioną w pierwszym wyzwaniu nazwę i pozostawiło Cię bez skonfigurowanej właściwości username! Jeśli tak się stało, w porządku, po prostu powtórz krok z pierwszego wyzwania w celu dodania swojej nazwy, a następnie wykonaj krok powyżej.

Nazwy GitHub i ta w konfiguracji Git nie pasują

Popularnym błędem jest wystąpienie różnic pomiędzy nazwą GitHub, a wprowadzoną w konfiguracji Gita. Na przykład, 'JLord' to nie to samo co 'jlord'

By zmienić ustawioną w git nazwę użytkownika, zrób to samo co poprzednio, tym razem z poprawną wielkością liter:

git config --global user.username <NazwaUżytkownika>

Kiedy to już zrobisz, zweryfikuj poprawność ponownie!
