Instalacja git na Twoim komputerze i skonfigurowanie Twojej nazwy oraz e-maila.

Git

Git jest darmowym oprogramowaniem o otwartym kodzie (kod źródłowy jest publicznie dostępny) napisanym przez Linusa Torvaldsa, który stworzył również jądro systemu operacyjnego Linux.

Git jest programem służącym do śledzenia zmian w czasie, znanym w programowaniu jako kontrola wersji. Jeśli używałeś funkcji śledzenia zmian w edytorach tekstowych to jesteś już zaznajomiony z tą koncepcją!

Instalacja git

Zalecamy zainstalowanie Git na Twoim komputerze poprzez pobranie aplikacji GitHub Desktop. Nie będziemy używać aplikacji okienkowej w Git-it (uczymy się obsługi z poziomu terminala!), ale zawiera ona Gita i jest najprostszym sposobem na zainstalowanie Gita na wszystkich platformach w ten sam sposób.

  • Windows: Użyj Git Shell jako aplikację terminala.
  • Mac: Otwórz GitHub Desktop i poprzez Preferences wybierz instalację wiersza poleceń. Korzystaj z aplikacji terminala.

Masz już Gita lub nie jesteś pewien? Wpisz git --version w aplikacji terminala i jeśli zwrócona zostanie wersja wyższa niż 1.7.10, jesteś gotów! Po więcej informacji odwiedź stronę Gita.

Oprogramowanie Git

Aplikacja GitHub Desktop umożliwia zrobienie wielu rzeczy za pomocą Gita lecz nie wszystko, dlatego też nauka obsługi Gita z poziomu wiersza poleceń jest ważna. Kiedy już to opanujesz, będziesz cieszyć się, że posiadasz aplikację okienkową, ponieważ organizuje ona informacje o projekcie w sposób bardziej zwizualizowany, jak strona GitHub.

Git sam w sobie nie jest jak inne programy na Twoim komputerze. Prawdopodobnie nie zobaczysz jego ikony na swoim pulpicie, ale zawsze będzie on dostępny i będziesz mógł z niego skorzystać w każdej chwili poprzez terminal lub aplikacje okienkowe Git.

Konfiguracja Gita

Kiedy aplikacja GitHub Desktop (a co za tym idzie również Git) zostanie zainstalowana, otwórz swój terminal. Możesz sprawdzić czy Git na pewno tam jest wpisując:

git --version

Spowoduje to zwrócenie wersji Gita zainstalowanej na Twoim komputerze, wyglądającej podobnie jak to:

git version 1.9.1

Następnie skonfiguruj Gita, by wiązał wykonywane czynności z Tobą:

Ustawianie nazwy:

git config --global user.name "Twoja nazwa"

Teraz skonfiguruj swój adres e-mail:

git config --global user.email "twójemail@example.com"

Symbol dolara w dokumentacji

Symbol dolara $ jest często używany w dokumentacji w celu podkreślenia, że linia stanowi treść polecenia (zobacz fragmenty kodu powyżej). Tak naprawdę nie jest konieczne wpisywanie znaku $ jako część polecenia, a jedynie tylko tego, co po nim następuje.
