GitHub Desktop

您可能在開始使用 Git-it 前就安裝過它了，但如果您還沒，請點選連結。

GitHub Desktop 應用程式讓您可以完成在終端機中的部份動作（雖然不是全部），而且有時候更容易閱讀。就我個人來說，我用它來檢視任何不小的diff。以這種方式看到您的變更真的很棒。

  • Github Desktop (OSX, Windows) 免費 by GitHub
  • Github Desktop 指南 by GitHub
